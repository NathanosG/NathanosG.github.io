Text Recognition Program

Utilising Python's image processing capabilities, I created a text recognition program which, given a written word input (which is created by the user drawing it with the mouse cursor), detects which letter has been drawn by comparing it's bitmap with the bitmaps of pre-known images. I made use of the "PyGame" package to create the canvas in which the user draws their characters on. I used an object oriented approach to this.
